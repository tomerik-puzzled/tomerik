A VIRTUAL ESCAPE GAME EXPERIENCE

The Professor OfThe Occult

Play our classic live-action escape game The Professor of the Occult via your computer in a virtual environment.
When you buy a session, you will receive an access code. Once you are ready to play, enter the code on the dedicated website. After activating, you have access for 48 hours. In order to play with your friends, set up a Zoom chat and give them the code and website link provided in the booking confirmation email.

Note: Game can only be played on a computer with a Chrome browser.
